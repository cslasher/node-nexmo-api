# Nexmo API Tests
[![StackShare](https://img.shields.io/badge/tech-stack-0690fa.svg?style=flat)](https://stackshare.io/cslasher/node-nexmo-api)
Node.js下的 Nexmo 簡訊發送與簡訊驗證

## 安裝
```shell
npm install
```

## 設定環境參數
* API_KEY
* API_SECRET

## 啟用
```shell
npm start
```
